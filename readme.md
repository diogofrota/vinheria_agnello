# Vinheria Agnello

Landing page institucional desenvolvida para representar digitalmente a **Vinheria Agnello**, com foco em uma experiência visual sofisticada, acolhedora e coerente com o posicionamento premium da marca.

## Preview

Projeto concebido como protótipo inicial para um site de vinheria, reunindo identidade visual refinada, navegação clara e estrutura preparada para futuras evoluções acadêmicas e funcionais.

## Objetivo

Este projeto foi criado como base para a construção de uma presença digital elegante para a Vinheria Agnello, buscando traduzir para o ambiente web elementos centrais da marca, como:

- curadoria especializada
- atendimento consultivo
- sofisticação visual
- experiência do cliente
- organização do catálogo

Além da proposta estética, o site também foi pensado como ponto de partida para evolução em sprints futuras, podendo receber integrações, novas páginas e recursos interativos.

## Funcionalidades da versão atual

A primeira versão do projeto contempla:

- topbar com informações institucionais
- cabeçalho com navegação principal
- menu responsivo para telas menores
- hero section com chamada principal
- área de destaque com diferenciais da solução
- seção “sobre a marca”
- seção de diferenciais
- vitrine inicial de rótulos
- área de experiência digital
- seção de avaliações
- formulário de contato
- rodapé institucional

## Estrutura do projeto

Nesta fase inicial, toda a aplicação foi construída em um único arquivo:

```bash
index.html


Esse arquivo reúne:

- marcação HTML
- estilos CSS internos
- comportamentos JavaScript internos

Essa abordagem foi adotada para facilitar a prototipação, a apresentação inicial e a validação visual do layout.

## Identidade visual

A proposta visual foi construída para transmitir elegância, tradição e sofisticação, com uma paleta inspirada em tons de vinho, dourado e creme.

| Dourado | `#c6a46a` |

Para reforçar a estética premium da marca, foram utilizadas as seguintes fontes:

- **Cormorant Garamond** — títulos e destaques

## Como executar

Para rodar o projeto localmente, basta abrir o arquivo `index.html` em um navegador.

### Opção 1: abertura direta

1. Baixe o arquivo `index.html`
2. Dê duplo clique sobre ele ou abra em seu navegador preferido

### Opção 2: VS Code com Live Server

1. Abra a pasta do projeto no VS Code
2. Instale a extensão **Live Server**
3. Clique com o botão direito em `index.html`
4. Selecione **Open with Live Server**
